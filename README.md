# Автоматическая сортировка почты

## Описание

Система автоматической классификации входящих писем. Программа читает текстовые файлы из папки `inbox`, определяет категорию письма по ключевым словам и копирует его в соответствующую папку внутри `results`.

## Категории

- Запросы на предоставление доступа
- Сообщения о внештатных ситуациях
- Вопросы по поводу оборудования
- Кадровые вопросы
- Информационные сообщения
- Спам
- Черновик
- Прочее
