# Автоматическая сортировка почты

## Описание

Система автоматической классификации входящих писем. Программа читает текстовые файлы из папки `inbox`, определяет категорию письма по ключевым словам и копирует его в соответствующую папку внутри `results`. В конце формируется HTML-отчёт со статистикой.

## Категории

- Запросы на предоставление доступа
- Сообщения о внештатных ситуациях
- Вопросы по поводу оборудования
- Спам
- Кадровые вопросы
- Информационные сообщения
- Черновик
- Прочее

## Установка и запуск

1. Склонируйте репозиторий: `git clone https://github.com/linalikaa/hackaton_team48.git`
2. Перейдите в папку: `cd hackaton_team48`
3. Запустите программу: `python main.py` (или `run.bat` на Windows, `./run.sh` на Linux/Mac)

## Тестирование

Выполните команду: `pytest test_classifier.py -v`
