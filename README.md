# Автоматическая сортировка почты

## Описание

Система автоматической классификации входящих писем. Программа читает текстовые файлы из папки `inbox`, определяет категорию письма по ключевым словам и копирует его в соответствующую папку внутри `results`. В конце формируется HTML-отчёт со статистикой.

## Категории
- Запросы на предоставление доступа
- Сообщения о внештатных ситуациях (ошибки, сбои)
- Вопросы по поводу оборудования
- Спам
- Кадровые вопросы
- Информационные сообщения (документы, счета, инструкции)
- Черновик (письма без отправителя)
- Прочее

## Как работает система

1. Программа сканирует папку `inbox`
2. Читает каждый `.txt` файл (с поддержкой кодировок utf-8 и cp1251)
3. Приводит текст к нижнему регистру и ищет ключевые слова
4. Определяет категорию и копирует письмо в `results/категория`
5. Выводит статистику в консоль и создаёт HTML-отчёт

## HTML-отчёт

После обработки всех писем создаётся файл `results/report.html`. Он содержит:

- Общее количество обработанных писем
- Список категорий с количеством писем в каждой
- Розовое оформление, курсив, удобную вёрстку
- Подпись с авторами проекта

Отчёт открывается в любом браузере и может быть передан руководству.

## Тестирование

Мы написали тесты для проверки классификации:

- Проверка каждого ключевого слова
- Проверка граничных случаев (пустые строки, только пробелы, случайные символы)
- Проверка длинных текстов с ключевыми словами в конце
- Проверка детерминированности (одинаковый результат при повторных вызовах)
- Реалистичные примеры писем из реальной выборки

Тесты запускаются командой `pytest test_classifier.py`. Все тесты проходят успешно.

## Bash-скрипт

Файл `run.sh` (Linux/Mac) и `run.bat` (Windows) запускают программу одной командой. Скрипт проверяет наличие Python и запускает `doc.py`.
